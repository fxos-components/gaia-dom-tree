## Installation

```bash
$ bower install gaia-components/gaia-dom-tree
```

## Examples

- [Example](http://gaia-components.github.io/gaia-dom-tree/)

## Tests

1. Ensure Firefox Nightly is installed on your machine.
2. `$ npm install`
3. `$ npm test`

If your would like tests to run on file change use:

`$ npm run test-dev`
